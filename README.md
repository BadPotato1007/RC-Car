# RC Car/Rover
This design was made in Tinker cad, and original comments that were placed on the design will not be seed in the file availible here on Github. If you would like to see more information, please open an issue, and ill add you as a contributer to the design under a copied file.
<br><br>
Most of the car is planned to be made out of wood, however many of the parts will be made from metal for bettery strength
<br>
Made by Neerav in 2026.
